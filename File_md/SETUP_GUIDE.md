# دليل التثبيت والتشغيل - ميزة الصور المصغرة

## المتطلبات الأساسية

- Node.js (v14 أو أحدث)
- npm أو yarn
- متصفح حديث

## خطوات التثبيت

### 1. تثبيت الـ Server

```bash
cd server
npm install
```

### 2. تثبيت الـ Client

```bash
cd Clint
npm install
```

## التشغيل

### 1. شغّل الـ Server

```bash
cd server
npm start
```

الـ Server سيعمل على: `http://localhost:3000`

### 2. شغّل الـ Client (في terminal منفصل)

```bash
cd Clint
npm start
```

الـ Client سيعمل على: `http://localhost:3000` (إذا كان port مختلفاً سيكون تلقائياً)

### 3. افتح المتصفح وانتقل إلى صفحة Recommended

```
http://localhost:3000/recommended
```

## الميزة الجديدة - الصور المصغرة

### ما هو الجديد؟

✨ تمت إضافة عمود جديد **"Photos"** في جدول الـ Recommended Cars يعرض:

1. **صورة مصغرة** لكل عربية
2. **Hover Effects** احترافية
3. **معرض صور تفاعلي** عند الضغط على الصورة

### كيفية الاستخدام

1. **اختر نطاق السعر:**
   - استخدم الـ Slider لاختيار السعر الأقصى
   - اضغط "Submit"

2. **اضغط على أي صورة مصغرة:**
   - ستُفتح نافذة معرض الصور الكامل
   - ستُعرض الصورة الأولى من المعرض

3. **تنقّل بين الصور:**
   - استخدم الأزرار ◀ و ▶
   - أو اضغط على الدوت أسفل الصورة
   - أو استخدم مفاتيح الأسهم (← →)

4. **أغلق المعرض:**
   - اضغط على زر X في الأعلى
   - أو اضغط خارج الصورة
   - أو اضغط Escape

## استكشاف الأخطاء

### المشكلة: الصور لا تُحمّل

**الحل:**
1. تأكد من وجود اتصال إنترنت
2. تأكد من أن Unsplash API متاح
3. جرّب متصفح مختلف
4. امسح الـ Browser Cache (Ctrl+Shift+Delete)

### المشكلة: الـ Modal لا يفتح

**الحل:**
1. تأكد من استيراد `CarImageModal` في `Recommended.js`
2. تأكد من استيراد `car-image-modal.css`
3. افتح Browser Console (F12) وابحث عن أخطاء
4. تأكد من أن الصور موجودة في البيانات

### المشكلة: Hover Effects لا تعمل

**الحل:**
1. تأكد من استيراد `car-thumbnail.css`
2. تأكد من عدم وجود CSS conflicts
3. جرّب في متصفح مختلف
4. امسح الـ Browser Cache

### المشكلة: الموبايل لا يعرض الصور بشكل صحيح

**الحل:**
1. تأكد من أن الـ responsive styles محملة
2. جرّب في وضع portrait/landscape
3. افتح الـ Developer Tools (F12)
4. اختبر على أجهزة مختلفة
